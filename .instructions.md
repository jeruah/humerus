# Instrucciones de Desarrollo - Proyecto Aproximación de Esfera en Húmero

## Requisitos del Proyecto

### Objetivo Principal
Aproximar una esfera a la semi-esfera articular de la cabeza del húmero con máxima fidelidad usando geometría diferencial.

### Requisitos Técnicos

1. **Lenguaje**: Python únicamente
2. **Entrada**: Archivos STL (modelos 3D de húmeros)
3. **Salida**: 
   - Centro de esfera aproximada (x, y, z)
   - Radio de esfera
   - Eje longitudinal del húmero
   - Métricas de fidelidad (error cuadrático medio, etc.)

### Metodología

#### Fase 1: Discretización
- Cargar modelo STL
- Crear malla triangular
- Discretizar superficie: samplear N puntos sobre la malla
- Identificar región articular de interés

#### Fase 2: Geometría Diferencial
- Calcular curvatura principal en cada punto discretizado
- Identificar región esférica (curvatura principal ≈ 1/radio)
- Usar curvatura como guía para localizar semi-esfera articular

#### Fase 3: Aproximación de Esfera
- Seleccionar X semillas aleatorias en región articular
- Para cada semilla:
  1. **Validación**: Verificar que está en región viable (curvatura semi-esférica)
  2. **Aproximación**: Usar algoritmo iterativo para encontrar esfera óptima
  3. **Auditoría**: Registrar cada paso, convergencia, métricas
- Refinar usando datos de convergencia anterior

#### Fase 4: Aproximación de Eje
- Calcular eje principal (PCA)
- Validar que eje pasa desde cabeza a distal
- Auditar trayectoria del eje

### Criterios de Éxito

1. ✅ Código completamente documentado
2. ✅ Sistema de auditoría funcionando en cada aproximación
3. ✅ Validación de viabilidad de semillas previa a aproximación
4. ✅ Manejo robusto de errores en convergencia
5. ✅ Convergencia demostrada con múltiples semillas aleatorias
6. ✅ Esferas aproximadas no divergen a otras partes del húmero

### Advertencias Comunes

⚠️ **Error de Convergencia Falsa**: Verificar siempre que la semilla está realmente en la región articular
⚠️ **Divergencia a Otras Superficies**: Usar restricciones de curvatura principal para evitar
⚠️ **Sensibilidad a Semillas**: Probar mínimo 10-20 semillas aleatorias por modelo
